✨ HOW IT WORKS
═══════════════════════════════════════════════════════════════════════════════

Your backend:

   1. Takes job description text
   2. Takes multiple resume PDFs
   3. Parses both using Gemini LLM API
   4. Extracts structured data (skills, experience, education)
   5. Matches skills using NLP (fuzzy matching + semantic similarity)
   6. Scores each resume in 3 sections:
      • Skills matching (50% weight)
      • Experience verification (30% weight)
      • Education matching (20% weight)
   7. Returns ranked candidates with detailed analysis

Final Score = (Skills × 0.50) + (Experience × 0.30) + (Education × 0.20)
